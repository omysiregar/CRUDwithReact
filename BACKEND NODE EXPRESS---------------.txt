BACKEND NODE EXPRESS-----------------------------------

buat pakage.json
> npm init -y

install bbrapa depencies

> npm i express mysql2 sequelize cors

sequelize : ORM atau object relational maping, untuk relasi db
cors : adalah Cors origin resource sharing , agar bisa diakses dari luar domain.



=====
-- buat file index.js , karena dari depedeni main nya di index.js

setelah set index.js 
install nodemon secara global 
> npm i -g nodemon
.................................
nodemon ini berguna untuk melihat aplikasi kita ini berjalan dengan baik apa tidak, dengan menjalan teminal:
> nodemon index
kenapa index ?, karna file main nya index.js atau bisa di bilang entry point nya
.................................
tapi kalau mau run pake
> npm run dev 
sebelum itu kamu harus set di json pakage 
> : 
"scripts": {
  "dev": "nodemon index.js"
}
